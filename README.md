# Mr. Fugga Blog 🐱

A simple **blog-style website** built while learning **HTML and basic web development**.

This project is inspired by my cat **Mr. Fugga** and was created as one of my first practice projects while learning how websites are structured and styled.

<br>

## 📌 Features

* Clean **dark mode UI**
* Navigation menu
* Blog-style post layout
* About section
* Contact section
* Image with caption
* Card-style blog posts

<br>

## 🛠️ Technologies Used

* **HTML5**
* **CSS3**
* **Google Fonts (Poppins)**

<br>

## 📚 What I Learned

While building this project I practiced:

* Structuring pages using **semantic HTML**
* Creating sections like **header, main, article, and footer**
* Designing layouts with **CSS**
* Using **CSS variables for theming**
* Building a simple **blog layout**

<br>
